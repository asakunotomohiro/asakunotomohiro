# Python言語
ここは、プログラミング言語のPython言語を勉強するブランチになる。  

## 開発環境
普段から`MacVim`を使っている。  
そのため、今回の勉強にもMacVimを使う。  
Windowsでもvimエディタを使うつもりだ。  

## 現在の環境
**pyenv**を使っている。  
それを今回覆す。  

```terminal
$ pyenv versions
  system
  3.8.6
* 3.9.7 (set by /Users/asakunotomohiro/.python-version)
$
```
しかし、vimエディタで上記バージョンを認識してくれず、Python2系で動くことを強要されている。  
私はPython3で動かしたいのに、端末との意識齟齬があり、しかもそれを解消できないでいる。  
ありがたいことに、pyenvは使わない方が良いという教示を得たため、[Homebrew](https://brew.sh/)でPython3のみを入れることにする。  

## pyenvからのアンインストール
そのまんま。

```terminal
$ pyenv versions
  system
  3.8.6
* 3.9.7 (set by /Users/asakunotomohiro/.python-version)
$ pyenv uninstall 3.8.6	←☆削除。
pyenv: remove /Users/asakunotomohiro/.pyenv/versions/3.8.6? [y|N]y
pyenv: 3.8.6 uninstalled
$ pyenv uninstall 3.9.7	←☆削除。
pyenv: remove /Users/asakunotomohiro/.pyenv/versions/3.9.7? [y|N]y
pyenv: 3.9.7 uninstalled
$ pyenv versions	←☆何もない(systemって元からあるの？)。
pyenv: version `3.9.7' is not installed (set by /Users/asakunotomohiro/.python-version)
  system
$
```
