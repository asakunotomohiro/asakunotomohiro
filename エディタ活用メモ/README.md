# エディタ活用メモ
普段利用しているエディタ利用で躓いた場合の回避策などを記録していく。  

## 利用エディタ一覧
以下、利用しているエディタ([dotfiles](https://github.com/asakunotomohiro/dotfiles)で説明済みだが)。  

* [Vim](https://www.vim.org)  
  [Windows用](https://www.kaoriya.net/software/vim/)・[Mac用](https://github.com/macvim-dev/macvim)  
* [neoVim](https://neovim.io)  
* [秀丸エディタ](https://hide.maruo.co.jp/software/hidemaru.html)  
* [サクラエディタ](https://sakura-editor.github.io)  


### Vimとは。
言わずと知れた世界に知れ渡ったエディタのひとつ。  
Unix・Linux・Windows・MacなどあらゆるOSに移植されている。  
いまは、AndroidOSでも使えるようになっている。  

[スクリプト](https://vim-jp.org/vimdoc-ja/usr_41.html)も使えるため、プログラミング言語のひとつとして基礎知識を調べた([勉強](../vimScript言語/README.md)した)。  

### Neovimとは。
vimエディタから派生したエディタ。  

### 秀丸エディタとは。
Windows専用のテキストエディタとして日本で有名になっている。  
しかし、有料なので、利用を躊躇している人もいる。  
金を払わずに社員に使わせている会社もあるぐらい有名で使いやすい。  

このエディタにも[マクロ](https://help.maruo.co.jp/be_hidemac/html/01_Index.html)が搭載されている。  

### サクラエディタとは。
Windows専用のフリーウェアのテキストエディタとして日本で有名なエディタのひとつ。  
そして、[オープンソース化](https://github.com/sakura-editor/sakura)されているようだ。  

このエディタにも[マクロ](https://sakura-editor.github.io/help/HLP000268.html)が搭載されている(リンク先は想像と違う)。  
**Windows Scripting Host([WSH](https://sakura-editor.github.io/help/HLP000269.html))** が使えるとのこと。  
他にも動かせる選択肢はあるようだが、そこまで食指は動かないかな・・・。  
何より、[サンプル](https://sakura-editor.github.io/help/HLP000270.html)を見る限り、使いこなしたいとは思わない感じなんだよな・・・。  

そのため、本当にテキストエディタとして使っている(秀丸エディタに無い機能があるから便利)。  
[正規表現](https://sakura-editor.github.io/help/HLP000089.html)では[秀丸エディタ](https://help.maruo.co.jp/hidemaru/html/190_CmdSearch_Regular.html)よりこっちのほうが[便利](../正規表現利用メモ/正規表現の勉強_作業メモなど何でも詰め込む.md)だと思っているのだが、どうだろう。
